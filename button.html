Button

About Button

There are 3 available kinds of button. The button as a standard button for all your triggering actions. The link button with role="button" for redirecting to an internal page of your site. The link button with target="_blank" will open in a new tab for redirecting to an external site by example. You can mix and select size, color and shape.

Button models

There is 3 button models, button as trigger and button trigger as link. 

                        
<button class="btn-s btn-purple" type="button">Button Trigger</button>
External Link 
                        
<a class="btn-s btn-purple" href="#" rel="noopener" target="_blank" role="button">External Link</a>
Internal Iink 
                        
<a class="btn-s btn-purple" href="#" rel="noopener" role="button">Internal Iink</a>

Button shapes

There is 3 button shapes, button class="btn-*", square class="btn-square-*" and circle class="btn-circle-*".
