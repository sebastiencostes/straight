Carousel

About Carousel

Carousels allow you to show multi content in one component. Generally pictures with or without text are provided. Pictures, text or both can be linked. Components are showed one by one.

Activation

To activate carousel, insert this line of code into your JavaScript file.

Carousel models

3 Carousel models are available. Simple carousels are provided. Each offers differents options about presentation. When using Carousel you must specify an id on each <yourtag id="yourCarouselID" class="carousel">. Thanks to srcset you can call your optimized media for specific device's breakpoint.

Carousel

Specify an ID in <div id="yourID" class="carousel">.

Carousel with text overlay

Title is linked.

Specify an ID in <article id="yourID" class="carousel carousel-overlay">.

Carousel with caption

Both image and title are linked.

Specify an ID in <article id="yourID" class="carousel carousel-caption">.
