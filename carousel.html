Carousel

About Carousel

Carousels allow you to show mutli content in one component. Generally picutres with or without text are provided. Pictures, text or both can be linked. Components are showed one by one.

Carousel models

Simple carousels are provided. There is 3 kinds purposed carousel. Each offers differents options about presentation. When using Carousel you must specify an id on each <yourtag id="yourCarouselID" class="carousel">. Thanks to srcset you can call your optimized media for specific device's breakpoint.
