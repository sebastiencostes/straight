About Header

Header represents introductory content, typically a group of introductory or navigational aids. It may contain some heading elements but also a logo, a search form, an author name, and other elements. Despite its name, it is not necessarily placed at the beginning of the page.

If you don't use an image to your header, check more about colors at color module.

Header models

Page Header with class <header class="header-presentation" role="presentation"> is designed to work with Nav. Feel free to add your image and more...

Page Header with background image with class <header class="header-presentation-background" role="presentation"> is also designed to work with Nav. Feel free to add your image and more...

Header with class <header class="header"> is a standalone.

Image header with class <header class="header-image bg-yellow"> is a standalone
