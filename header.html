About Header

Header represents introductory content, typically a group of introductory or navigational aids. It may contain some heading elements but also a logo, a search form, an author name, and other elements. Despite its name, it is not necessarily placed at the beginning of the page.

If you don't use a background image, check more about colors at color module.

Header models

Page Header with class <header class="presentation" role="presentation"> is designed to work with Nav. Header with class <header class="header"> is a standalone. Header article with class <header class="header-article"> works inside an article.
