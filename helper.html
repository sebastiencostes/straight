helper

About Helper

Helper module give you some extra classes for different cases of use. In Straight framework 1rem = 10px.

border

border classes works with solid border style at 1px width. if you need to add or remove border these classes are available. if you need colored borders go to color module.

no border

  • no border : class="no-b"
  • no border top : class="no-b-top"
  • no border right : class="no-b-right"
  • no border bottom : class="no-b-botto"
  • no border left : class="no-b-left"

border & border direction

  • border : class="border"
  • border top : class="b-top"
  • border right : class="b-right"
  • border bottom : class="b-bottom"
  • border left : class="b-left"

border radius

  • no border radius : class="no-b-radius"
  • smooth border : class="b-smooth"
  • border radius small : class="b-radius-s"
  • border radius medium : class="b-radius-m"
  • border radius large : class="b-radius-l"
  • border radius extra large : class="b-radius-xl"
  • circle : class="circle"

box shadow

if you want to play around with box shadow check these classes.

  • no box shadow : class="no-b-shadow"
  • box shadow standard : class="b-shadow"
  • box shadow hovered : class="b-shadow-hovered"
  • box shadow S : class="b-shadow-s"
  • box shadow M : class="b-shadow-m"
  • box shadow L : class="b-shadow-l"
  • box shadow XL : class="b-shadow-xl"
  • box shadow inset : class="b-shadow-inset"

display

if for some reason you have to change the displaying of your element check these classes.

  • display none : class="d-none"
  • display block : class="d-block"
  • display inline block : class="d-inline-block"
  • display flex : class="d-flex"
  • display inline flex : class="d-inline-flex"
  • display grid : class="d-grid"
  • display inline grid : class="d-inline-grid"

flexbox

Your displaying content will thank you. Your parent container must be in Flex displaying or in Grid displaying.

flex direction

  • row direction : class="f-dir-row", the native flex direction.
  • row reverse : class="f-dir-row-reverse"
  • column direction : class="f-dir-col"
  • column reverse direction : class="f-dir-col-reverse"

flex wrap

  • wrap : class="f-wrap"
  • no wrap : class="f-nowrap"
  • wrap reverse : class="f-wrap-reverse"

justify content

  • justify between : class="j-between"
  • justify evenly : class="j-evenly"
  • justify center : class="j-center"
  • justify around : class="j-around"
  • justify stretch : class="j-stretch"
  • justify start : class="j-start"
  • justify end : class="j-end"
  • justify revert : class="j-revert"

align items

  • align baseline : class="i-base"
  • align revert : class="i-revert"
  • align center : class="i-center"
  • align stretch : class="i-stretch"
  • align start : class="i-start"
  • align end : class="i-end"

align self

  • self center : class="self-center"
  • self stretch : class="self-stretch"
  • self flex start : class="self-f-end"
  • self flex end : class="self-f-start"
  • self start : class="self-end"
  • self end : class="self-start"

flex basis, values are in % from 1 to 100.

  • flex basis : class="f-basis-*"

order, values from 1 to 16 are available.

  • order : class="order-*"

height

height works with different kinds of values. height values can be relative to container, screen & global. height relative to container works with value in %, relative to screen with vw and global with rem.

percentage height relative to container works with values from 1 to 100.

  • height in percentage : class="h-prc-*"
  • max height in percentage : class="max-h-prc-*"
  • min height in percentage : class="min-h-prc-*"

screen height relative to screen works with values from 1 to 100.

  • height in vh : class="h-vh-*"
  • max height in vh : class="max-h-vh-*"
  • min height in vh : class="min-h-vh-*"

global height works with values from 1 to 300.

  • height in rem : class="h-rem-*"
  • max height in rem : class="max-h-rem-*"
  • min height in rem : class="min-h-rem-*"

margin

if you need external margin, these classes are available.

zero & auto

  • margin zero : class="m-0"
  • margin top zero = class="m-t-0"
  • margin right zero class="m-r-0"
  • margin left zero class="m-l-0"
  • margin bottom zero class="m-b-0"
  • margin horizontal axe auto class="m-x-auto"
  • margin vertical axe auto class="m-y-auto"
  • margin top auto class="m-t-auto"
  • margin right auto class="m-r-auto"
  • margin bottom auto class="m-b-auto"
  • margin left auto class="m-l-auto"

margin works with values from 1 to 16. Units are in rem.

  • margin : class="m-*"

margin direction works with values from 1 to 16. Units are in rem.

  • margin top : class="m-t-*"
  • margin right : class="m-r-*"
  • margin bottom : class="m-b-*"
  • margin left : class="m-l-*"

opacity

opacity works with value from 0 by 0X to 1.

  • opacity 0 : class="op-0"
  • opacity X : class="op-0X"
  • opacity 1 : class="op-1"

padding

if you need internal "margin", these classes are available.

padding works with values from 1 to 16. Units are in rem.

  • padding zero : class="p-0"
  • padding : class="p-*"
  • padding top : class="p-t-*"
  • padding right : class="p-r-*"
  • padding bottom : class="p-b-*"
  • padding left : class="p-l-*"

position

if for some reason you have to change the positioning of your element check these classes.

  • position absolute : class="pos-absolute"
  • position absolute center : class="pos-absolute-centering"
  • position relative : class="pos-relative"
  • position sticky : class="pos-sticky"
  • position fixed : class="pos-fixed"
  • position fixed top : class="pos-fixed-top"
  • position fixed right : class="pos-fixed-right"
  • position fixed left : class="pos-fixed-left"
  • position fixed bottom : class="pos-fixed-bottom"

width

width works with different kinds of values. Width values can be relative to container, screen & global. width relative to container works with value in %, relative to screen with vw and global with rem.

percentage width relative to container works with values from 1 to 100.

  • width in percentage : class="w-prc-*"
  • max width in percentage : class="max-w-prc-*"
  • min width in percentage : class="min-w-prc-*"

screen width relative to screen works with values from 1 to 100.

  • width in vw : class="w-vw-*"
  • max width in vw : class="max-w-vw-*"
  • min width in vw : class="min-w-vw-*"

global width works with values from 1 to 300.

  • width in rem : class="w-rem-*"
  • max width in rem : class="max-w-rem-*"
  • min width in rem : class="min-w-rem-*"

z-index

z-index works with value from 1 to 100.

  • z-index : class="z-*"
