About Image

Straight Framework image... attribute alt="" must be present on each <img src="..." alt=""> tag.

responsive image

Work well on devices with widely different screen sizes, and other such features. To use it just add class="responsive-img". Resize your screen to see it in action. 

                
<img class="responsive-img" src="pathTo/img.png" alt="">

rounded image

if you need rounded image just add class="rounded-img-*" or class="rounded-img-smooth" or class="circled-img" , rounded images are responsive too. 4 rounded sizes are available s, m, l and xl. 

                
<img class="rounded-img-smooth" src="pathTo/img.pn" alt="">
<img class="rounded-img-s" src="pathTo/img.pn" alt="">
<img class="rounded-img-m" src="pathTo/img.pn" alt="">
<img class="rounded-img-l" src="pathTo/img.pn" alt="">
<img class="rounded-img-xl" src="pathTo/img.pn" alt="">
<img class="circled-img" src="pathTo/img.png" alt="">

srcset

srcset defines the set of images we will allow the browser to choose between, and what size each image is. It's a good way for website optimization. At each media="(min-width: ***px)" attribute the same image but not the same file at different size is loaded. Each image have different file size. It permits from mobile to desktop displaying optimization. 

                
<picture>
    <source srcset="/yourpathTo/XLarge.png" media="(min-width: 1400px)">
    <source srcset="/yourpathTo/Large.png" media="(min-width: 992px)">
    <source srcset="/yourpathTo/Medium.png" media="(min-width: 728px)">
    <img class="responsive-img" src="/yourpathTo/defaultORmobile.png" alt="">
</picture>

element represents self-contained content, potentially with an optional caption, which is specified. The figure, its caption, and its contents are referenced as a single unit. If you want your title in 1st position just inverse the <picture> and <figcaption> position.
