About Image

Straight Framework image module, discover different kinds of process for displaying your images as images. the attribute alt="" must be present on each <img src="..." alt=""> tag. It must be specified only if it adds some crucial info.

responsive image

Work well on devices with widely different screen sizes, and other such features. It will preserve your aspect image ratio. To use it just add class="responsive-img". Resize your screen to see it in action.

rounded image

if you need rounded image just add class="rounded-img-*" or class="rounded-img-smooth" or class="circled-img" , rounded images are responsive too. 4 rounded sizes are available s, m, l and xl.

srcset

srcset defines the set of images we will allow the browser to choose between, and what size each image is. It's a good way for website optimization. At each media="(min-width: ***px)" attribute the same image but not the same file (or different image too) at different size is loaded. Each image have different file size. It permits from mobile to desktop displaying optimization. Resize your screen to see it in action.

image with caption

element represents self-contained content, potentially with an optional caption, which is specified. The figure, its caption, and its contents are referenced as a single unit. If you want your title in 1st position just inverse the <picture> and <figcaption> position.

Same width/height ratio images gallery

Images gallery with the same width / height ratio.

images gallery with different width/height ratio

Images gallery with different widths / heights ratio. Int this case the gallery will auto detect if your images are portrait, landscape or square orientation. All the images will be center positionned and they will cover their own space without any width/height ratio deformation. Just add <div class="image-gallery"> to the container.

you can learn more about layout at layout module.

No column and no row images gallery with different width/height ratio

The same as precedent, with no column and no row. Just add by <div class="g-no-row-col"> on the container.

image lightbox

Highlighted images permit you to scale up your pictures, artworks, projecys and more to visitors. Just add class="image-highlight" to your <img class="..." src="..." alt=""> tag. To scale up and scale down image you have just to clicking on images themselves just below.
