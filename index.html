About Straight framework

Version BETA 0.3

Mobile first and responsive framework based on HTML5, Flexbox, CSS Grid with used of relative units and JS. You just have to click & paste code you want and voilà. It works with modern browsers (Chrome, Firefox, Safari, Edge with chromium engine, Opera) and updated versions of theses one. According to microsoft, Internet Explorer 11 will be retired on June 15, 2022 . You can view more details at the following link. Straight framework is not suitable for any version of IE or older version of EDGE without chromium engine.
