Layout

About Layout

built around a 16 columns system, create your own layout. CSS grid and flexbox are used to allow you a great ratio between easiness to use and standard modern web displaying possibilities. In Straight framework 1rem = 10px and in most cases containers are in grid and contents are in flex. There is no flaoted element. Lot of helpers classes come to rescue too, you can check more at Helper module. Of course Straight framework is Mobile Friendly.

Container

4 containers are available. 1 fixed, 1 relative to screen width, 1 relative to container width and 1 full width relative to container. 3 of them are spaced to left and right. In the example below you can see difference between each relativity. Click on the button just below to see variation.

Breakpoint

Straight framework includes 5 default breakpoints, from s to xxl. These breakpoints refers to some devices screen width, from mobile to desktop screen. they don’t target every use case or device.

S breakpoint ≥ 280px is phone

M breakpoint ≥ 728px is tablet portrait

L breakpoint ≥ 992px is tablet landscape

XL breakpoint ≥ 1400px is laptop

XXL breakpoint ≥ 1880px is desktop

Grid

16 columns grid system based, 16 columns evenly offers more displaying opportunities on height / length screen ratio. Grid container and container explained earlier are 2 different things. First, when you create your own grid system you must add class="g-col" into that will be your grid container. Thanks to breakpoints you can create different grids at each of them. Inner grid system works with class="g-col g-col-*-*", in this case asterisks represents breakpoint and number of columns. If you want 3 columns at landscape tablet breakpoint just add class="g-col g-col-l-3" and if you want too 6 columns at desktop breakpoint just add class="g-col g-col-l-3 g-col-xxl-6" to the same grid container. Resize your window to see the grids in action.
