Layout

About Layout

built around a 16 column system, create your own layout. CSS grid and flexbox are used to allow you a great ratio between easiness to use and standard modern web displaying possibilities. Lot of helpers classes come to rescue too, you can check more at Helper module. Of course Straight framework is Mobile Friendly.

Container

4 containers are available. 1 fixed, 1 relative to screen width, 1 relative to container width and 1 full width relative to container. 3 of them are spaced to left and right. In the example below you can see difference between each relativity. Click on the button just below to see variation.

Grid

16 columns grid system based, 16 columns evenly offers more displaying opportunities on height / length screen ratio. In Straight framework 1rem = 10px and in most cases containers are in grid and contents are in flex. There is no flaoted element. Resize your window to see the grid in action.
