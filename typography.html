typography

About Typography

Straight framework mostly works with CSS clamp property and rem units to sizing text for optimum text rendering on every device and OS. System font stack is applied.

If you need to add colors to your text see color module.

heading

The <h1> to <h6> elements represent six levels of section headings. <h1> is the highest section level and <h6> is the lowest. Using more than one <h1> is allowed by the HTML specification, but is not considered a best practice. Using only one <h1> is beneficial for screenreader users and SEO. Heading classes are also available. You can add some classes to align your text and more...

icon

icons are small pictogram representing an action, an object, a software, a type of file and more. In Straight framework inner icons are SVG formated.

These icons come from ionic ionicons. Check out more to find icon you need.

Icon colors

This is the same color class used for text color. Just use class="t-*".

Icon sizes

6 sizes are purposed, from s to xxxl. For each size just use class="icon-*". Icon with icon-s is the default size.

The <address> element indicates that the enclosed HTML provides contact information for a person or people, or for an organization. No heading, sectioning, header or footer are allowed.

mark

The <mark> element represents text which is marked or highlighted for reference or notation purposes, due to the marked passage's relevance or importance in the enclosing context. Just add <mark class="mark-*">.
