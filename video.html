About Video

embeds media player which supports video playback into the document.

Embedded video

The controls attribute adds video controls, like play, pause, and volume. The video class allows the <video> to be responsive. The <source> element allows you to specify alternative video files which the browser may choose from. The browser will use the first recognized format. The text between the video and video tags will only be displayed in browsers that do not support the video element.

Currently three video formats are supported by the HTML5 specification:

  • WebM
  • Ogg
  • MP4

Attributes :

  • autoplay : play automatically
  • controls : play/pause, volume, full-screen, seek slider
  • controlsList : nofullscreen, nodownload, noremoteplayback
  • loop : video played in a loop
  • muted : mute the sound
  • poster : poster="pathTo/img.png"
  • preload : none, metadata, auto

Accessibility & subtitles

For achieving this you will need a .vtt file. Format for displaying timed text tracks alongside video using <track>.

<track> attributes :

  • default : enabled automatically when the video starts playing
  • kind : kind of text track provided
  • label : defines the title of the text track
  • src : path to your .vtt file
  • srclang : language the track is in

kind values :

  • captions : overlays transcription of audio onto the video
  • chapters : titles for different portions
  • descriptions : overlays a full audio description
  • subtitles : overlays written translations
